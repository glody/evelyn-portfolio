# Evelyn’s portfolio

Static personal portfolio site (HTML & CSS). Open `index.html` in a browser locally, or host on [GitHub Pages](https://pages.github.com/).

## GitHub Pages

After pushing this repo to GitHub: **Settings → Pages → Build and deployment → Source**: Deploy from branch `main`, folder `/ (root)`.
